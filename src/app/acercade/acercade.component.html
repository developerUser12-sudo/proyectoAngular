Acerca de esta aplicación

Bienvenido a nuestra aplicación de Busqueda de Películas y Series. Esta aplicación te permite buscar películas y series de diferentes géneros, años y otros criterios utilizando la API de OMDB

¿Cómo funciona?

Solo tienes que ingresar el nombre de la película o el año en el campo de búsqueda, y la aplicación te devolverá los resultados de la API

Características:

  • Búsqueda por nombre de película
  • Búsqueda por año de estreno
